# VRC_Unity_tutorial 編集ルール

このリポジトリはVRChatアバター改変の汎用ナレッジベース。LLMのRAGソースとして万人が使う前提で書く。

## KB追記

- 追記先は vrchat-avatar-kb/index.md の表で判断する。新カテゴリファイルを勝手に作らない
- 各ファイルの共通構成(概要/依存関係/対応する改変パターン/改変時の注意点/Quest対応時の注意/関連ファイルパス/よくあるトラブル/関連ツール/バージョン履歴)の該当セクションに入れる
- 特定アバター・特定プロジェクト固有の記述は一般化してから書く(固有名は例示に留める)
- 根拠を意識する。ソースコード/公式ドキュメント由来か、実測由来か。実測由来は「実測」と明記する
- commit message: "kb: <一行要約>"
- push はリポジトリオーナーのみが行う
